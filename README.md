# TestCop

## Web GUI Application

This project is a Web GUI application built using .NET 8. The application allows users to input two values and perform arithmetic operations (addition, subtraction, multiplication, and division) on them. If the input values are not numeric, the application will display 'ðŸ™…'.

## How to Run the Application

1. Clone the repository:
   ```
   git clone https://github.com/TomoharuMisawa/TestCop.git
   ```
2. Navigate to the project directory:
   ```
   cd TestCop/WebApp
   ```
3. Build the project:
   ```
   dotnet build
   ```
4. Run the project:
   ```
   dotnet run
   ```
5. Open a web browser and navigate to `http://localhost:5000` to use the application.
